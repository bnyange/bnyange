## Projects I'm proud of

⚽ Football Match Prediction Engine *(ongoing)*
A high-fidelity prediction system built to beat bookmaker baselines across the
Top 5 European Leagues. This is the project I'm most invested in right now.

**How it works:**
- **Stacked Ensemble Model** — XGBoost + Random Forest combined with a Logistic
  Regression meta-learner for calibrated match outcome probabilities
- **Tactical Absence Engine** — parallel Transfermarkt scraper that tracks
  injuries/suspensions and calculates individual player PPG impact on results
- **Poisson Score Matrix** — generates precise probabilities for the top 5 most
  likely exact scorelines per fixture
- **Market Sentiment Integration** — real-time API connectors for Kalshi and
  Polymarket to detect crowd bias and smart money shifts
- **Hyperparameter Self-Tuning** — automated Grid Search minimising Log Loss on
  every retrain cycle

The system runs on a Research → Strategy → Execution lifecycle, identifying value
by comparing a match's true clinical probability against the market's implied odds.
Built entirely in Python with Pandas, NumPy, XGBoost, and Scikit-learn in
Jupyter Notebooks.
